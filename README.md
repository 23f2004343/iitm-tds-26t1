## 📋 Scope

This repository contains graded assignments and projects completed as part of the TDS course.  
Emphasis is placed on:

- Reproducible workflows
- Version control discipline
- API development
- Data processing pipelines
- Deployment and CI/CD automation

## 🛠 Tech Stack

- **Development**: VS Code, Git, Docker, GitHub Actions, Dependabot, GitHub Pages, GitHub Codespaces, DevContainers, ngrok, cloudflared, localtunnel, uv, pip, curl, HTTPie, wrangler, Bash, PowerShell
- **Backend**: FastAPI, Flask, Starlette, Uvicorn, Gunicorn, Pydantic, python-multipart, itsdangerous, httpx, requests, python-dotenv, SSE, StreamingResponse
- **Data / ML**: pandas, NumPy, scikit-learn, Pillow, hashlib, AST, pickle, csv
- **Databases**: PostgreSQL, Redis, SQLite
- **Frontend / Deployment**: HTML, CSS, JavaScript, Vercel, Cloudflare Workers, Railway, Render, Hugging Face Spaces, Google OAuth 2.0, JWT
- **Testing**: pytest, httpx
- **AI / LLM**: OpenAI API (GPT-4o-mini, text-embedding-3-small), Google Gemini (gemini-2.0-flash-exp), Ollama (llama3), Groq Whisper v3, AIPipe, OpenRouter, VADER
- **Scraping / Media**: Playwright, yt-dlp, FFmpeg, Google Sheets
